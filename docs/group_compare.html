Controls
Loading data
Upload folder Select a directory of JSON result files. Each file is parsed for group, theta, tests, and checkpoint fields.
Sidebar
Groups Groups are derived from the group field in each JSON. Files sharing the same group name are treated as one run at different training point counts (n).
Prefix sections Groups are visually organised by their alphabetic prefix (e.g. al, sobol). Click a section header to expand or collapse it. Use all / none to bulk-select within a prefix.
Filter Type to filter visible group names.
n range Shown next to each group name. Derived from the length of the theta array in each JSON.
Controls bar
Dims Toggle individual output dimensions on or off. The number of pills updates automatically to match the selected metric's dimensionality.
Result Dropdown to select which test result to display. Options are grouped by type — metrics (nrmse, mse) and parameters (parameter_lengths, spatial_lengths). Only results present in the loaded data appear.
View Toggle between chart (line plots of metric vs n) and table (one row per group at a selected n).
Y scale Switch the chart y-axis between linear and log scale. Disabled in table view.
Table view
Training points slider Select a specific n value. One row per selected group is shown at that n; groups with no data at that exact n display —.
Best value highlight The lowest value in each dimension column is highlighted in green.
